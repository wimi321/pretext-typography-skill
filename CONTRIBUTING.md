# Contributing

## Local Loop

Install dependencies only if you need the npm script runner:

```sh
npm install
```

Validate the repo structure:

```sh
npm run validate
```

Preview the showcase locally:

```sh
npm run showcase
```

Open `http://127.0.0.1:4173/showcase/`.

## What To Improve

Good contributions include:

- stronger Pretext-driven templates
- clearer multilingual layout guidance
- better demo patterns for editorial UI
- tighter validation for skill metadata and assets
- improved README examples in English or Chinese

## Content Principles

- keep `SKILL.md` action-oriented
- keep references concise and easy to scan
- prefer reusable templates over abstract explanation
- preserve attribution to the upstream Pretext project

## Pull Requests

- explain the user problem the change helps solve
- include screenshots or a short note if the showcase changes
- run `npm run validate` before pushing
